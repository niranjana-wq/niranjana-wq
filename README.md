## 🧠 About Me

AI/ML Engineer focused on building **real-time intelligent systems** using Computer Vision, NLP, and LLMs.  
I work on transforming models into **production-ready applications** with scalable pipelines and APIs.

---

## 🚀 Live Systems

🧠 **CareCaller AI**  
→ LLM-based healthcare assistant with voice interaction  
→ Multi-turn context + document-aware inference  

🎓 **Student Engagement Analyzer**  
→ Real-time attention tracking using CV + LSTM  
→ Live feedback using OpenCV + MediaPipe  

🫁 **Pneumonia Detection System**  
→ Deep learning model for medical imaging  
→ Focused on healthcare diagnosis  

🎙 **Multi-Modal AI Interaction System**  
→ Voice + NLP + real-time processing pipeline  
→ Handles edge cases like escalation & scheduling
